O que você sabe sobre Web design?

O web design é o estilo/design/layout que uma página web possui, ele funciona em páginas em HTML ou em outras linguagens, ele tem que possuir esses 4 pontos fundamentais: responsividade, Escaneabilidade, tipografia e velocidade de carregamento. Sobre os três temas:

design linguagem visual criação de identidade visual
Um design é uma atividade responsável pelo planejamento, criação e desenvolvimento de produtos e serviços. Na web o design interage em diversas modalidades e áreas da web como: Páginas de site ou blog, e-commerce, redes sociais, E-mail entre outros. A linguagem visual é a parte da semiótica/semiologia que se utiliza de signos não verbais para expressar uma informação.
A Identidade visual é um conjunto de elementos que representam graficamente alguma marca. Ela é muito importante para que os clientes e o público de uma maneira geral possam distinguir uma empresa, produtos e serviços de outras marcas.

Confira 5 passos para a criação de identidade visual:

1. Defina em primeiro lugar qual a sua mensagem
2. Estabeleça uma relação emocional
3. Trabalhe com a simplicidade na criação de uma identidade visual
4. Mantenha sua identidade visual em toda sua comunicação
5. Crie um manual da marca
